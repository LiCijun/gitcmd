Detailed Description

Abstract class for modeling errors within CoreASM specifications.

It stores the attributes of an error as key/value pairs. The class also offers the serialization of the attributes into a String, which is then used as an attribute for an error marker. Afterwards the String can be deserialized back into an instance of this class.

This class does not manage what attributes are available, this is the task of its subclasses.
