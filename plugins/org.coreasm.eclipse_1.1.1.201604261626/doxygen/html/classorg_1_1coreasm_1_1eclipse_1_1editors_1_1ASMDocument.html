void org.coreasm.eclipse.editors.ASMDocument.checkIfSpecificationIsIncluded ( )
private

This method checks if the CoreASM specification which is represented by the document is a normal specification (starting with the keyword "CoreASM") or if it is intended to be included in other specifications (starting with the keyword "CoreModule").

The result of the method is stored in the variable isIncluded.
