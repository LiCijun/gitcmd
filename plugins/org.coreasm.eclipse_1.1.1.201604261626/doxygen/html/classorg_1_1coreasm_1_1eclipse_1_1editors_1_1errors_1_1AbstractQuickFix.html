boolean org.coreasm.eclipse.editors.errors.AbstractQuickFix.checkValidility ( AbstractError  error)

Checks if the QuickFix can be applied for the given error.

The default implementation of AbstractQuickFix always returns true. Subclasses must override this method if it depends on the error object if a QuickFix can be applied to the error.

void org.coreasm.eclipse.editors.errors.AbstractQuickFix.initChoices ( AbstractError  error)

This method initializes the choices which are offered to the hover window.

Subclasses which offer choices must override this method. The implementation of the method in this class does nothing, so the list of choices remains empty.
