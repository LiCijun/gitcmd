void org.coreasm.eclipse.editors.ASMEditor.documentChanged ( DocumentEvent  event)

This method is called each time the document is edited.

It ensures that the document is parsed after it has been modified. However, the parser is not run after each edit, it is just rescheduled for a second after the last edit. So the document won't be parsed unless the user makes no input for at least one second.
