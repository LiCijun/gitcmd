void org.coreasm.eclipse.editors.outlining.AbstractContentPage.selectionChanged ( SelectionChangedEvent  event)

This method is called if the user clicks onto a node of the outline tree.

In this case, the editor should highlight the region which the clicked node refers to. This method retrieves the selected element from the tree and delegates it to the abstract method setHiglighting.
